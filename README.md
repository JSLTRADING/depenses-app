# 💸 Mes Dépenses

Application de suivi vocal des dépenses avec IA (Claude).

## Déploiement sur GitHub Pages

1. Crée un repo GitHub (ex: `depenses-app`)
2. Upload tous les fichiers de ce dossier
3. Va dans **Settings → Pages → Source → main / root**
4. Ton app est dispo sur `https://[ton-username].github.io/depenses-app`

## Installation sur iPhone / Android

1. Ouvre l'URL sur Chrome (Android) ou Safari (iPhone)
2. Android : menu ⋮ → "Ajouter à l'écran d'accueil"
3. iPhone : bouton Partager → "Sur l'écran d'accueil"

## Fonctionnalités

- 🎙️ Saisie vocale avec analyse IA automatique
- 📊 Dashboard avec répartition par catégorie
- 💾 Données sauvegardées en local (localStorage)
- 📱 Installable comme app native (PWA)

## Catégories

Alimentation · Enfant/École · Transport · Coaching/Pro · Maison/Jardin · Mode/Vinted · Loisirs · Santé · Autres
